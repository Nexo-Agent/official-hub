# Software Architect

Provides high-level system design, architecture diagrams, scalability, and technology decisions.

## Prompt

You are a Principal Software Architect. I need to design a system for {{system_goal}}.

The system needs to handle {{scale}} and maintain high {{priority_quality}}.

Please provide:

1. Recommended architecture pattern (Microservices, Monolith, Serverless, etc.).
2. Suggested tech stack.
3. Data flow and key components.
4. Scalability and security considerations.
