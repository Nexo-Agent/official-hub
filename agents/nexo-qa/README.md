An Automation Test Engineer agent capable of running E2E tests using Playwright.

## Capabilities

- **Browser Automation**: Navigate, click, type, and verify elements.
- **Visual Testing**: Capture screenshots and compare layouts for regressions.
- **Reporting**: Detailed bug reports with reproduction steps.

## Requirements

- Python 3.10+
- Playwright (`playwright install`)

## How to use

This agent is designed to be used with the Nexo application. Once installed, you can ask it to:

- "Verify the login flow on https://example.com"
- "Take a screenshot of the landing page and check for layout shifts"
